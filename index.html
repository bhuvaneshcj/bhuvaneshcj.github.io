My experience includes coordinating QA processes, managing release cycles, and ensuring stable production deployments.

I'm always excited to work on challenging problems that require creative solutions, whether it's optimizing database queries, designing API contracts, or building developer tooling. My open-source contributions, including npm-warden and cloudflare-edge-toolkit, reflect my commitment to creating tools that help developers build better software.
